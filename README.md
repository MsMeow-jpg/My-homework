# Selector Specificity Example

Цей проект демонструє роботу специфічності CSS-селекторів на прикладі простого веб-сайту з двома сторінками: головною (`index.html`) та додатковою у папці `pages` (`newfolder.html`).

---

## Опис проекту

- Веб-сайт з прикладами різних селекторів CSS і їхньою вагою.
- Кожна сторінка має власний HTML-файл.
- Для стилізації використовується один спільний файл `style.css`.
- Навігація між сторінками здійснюється за допомогою посилань.

---

## Функціональні можливості

- Показ різних типів CSS-селекторів та їх пріоритетів (класи, ідентифікатори, атрибути, псевдокласи).
- Демонстрація впливу селекторів на зовнішній вигляд елементів (фон, колір тексту).
- Структура проекту підтримує розміщення сторінок у підпапках.
- Адаптивний і чистий код для подальшого розширення.

---
